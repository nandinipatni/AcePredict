# Roadmap

## Current Phase

Version 1 (Documentation Foundation)

Status: In Progress

Goal:

Establish a professional repository structure, project documentation, and architectural foundation before implementation begins.

---

## Phase 1: Foundation

Deliverables:

* Repository setup
* Documentation structure
* Architecture planning
* Database design
* Feature planning
* Engineering decision records (ADRs)

Status: In Progress

---

## Phase 2: Data Layer

Deliverables:

* ATP dataset acquisition
* Data validation workflow
* Data cleaning pipeline
* Dataset documentation

Status: Planned

---

## Phase 3: Feature Engineering

Deliverables:

* Elo rating calculation
* Ranking-based features
* Surface performance features
* Head-to-head statistics
* Recent form metrics

Status: Planned

---

## Phase 4: Machine Learning

Deliverables:

* Baseline model
* Model evaluation
* Prediction workflow

Status: Planned

---

## Deferred

The following items are intentionally excluded from Version 1:

* FastAPI
* Next.js
* Deployment
* Authentication
* Live ATP APIs
* SHAP explanations
* Tournament simulations
* CI/CD pipelines
* Docker

These features may be considered after a stable prediction pipeline exists.
